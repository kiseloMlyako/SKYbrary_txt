Background
This safety enhancement increases the potential safety impact of Safety Enhancement 1, Terrain Avoidance Warning System (TAWS) by developing procedures to include Global Positioning System (GPS) sensors for TAWS, and to ensure that updates to terrain databases, alerting algorithms, and new options to TAWS are incorporated as soon as possible.
 
Related Implementation Plan
Detailed Implementation Plan
 
Related Outputs
Airbus Industries, Boeing, Embraer, and Bombardier Aerospace have committed to install or provide the option to install GPS sensors on all current production model airplanes and all new type designs.
