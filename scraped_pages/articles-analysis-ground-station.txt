Definition
Analysis Ground Station (AGS) - Flight data analysis TOOL Developed by SAGEM (France).
Description
Aim is to provide report generation from automatic and manual data selection for FOQA (Flight Operational Quality Assurance) and the closely related MOQA (Maintenance Operational Quality Assurance), import/export functions, numerous expanded programming capabilities, advanced analysis, and database management features. In an automatic analysis AGS can analyse and process all data available from a recorder in order to provide a customised report as requested.
